# EasyExpense

A modern **Expense Tracker** web application built with **React, TypeScript, Vite, and Tailwind CSS**.  
It allows users to track income and expenses, categorize transactions, view monthly summaries, and visualize data using interactive charts.

---

## âœ¨ Features

- Add income and expense transactions
- Categorize transactions (Food, Transport, Bills, etc.)
- Persistent data storage using **LocalStorage**
- Monthly grouping of transactions
- Summary dashboard:
  - Total income
  - Total expenses
  - Current balance
- Category-based **doughnut chart**
  - Toggle between **Income** and **Expense**
  - Percentages shown in and legend
  - Consistent color mapping per category
- Delete transactions
- Clean and responsive UI
